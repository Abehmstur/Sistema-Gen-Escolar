\doxysubsection{Descrição detalhada}
Representa um funcionário que é uma pessoa e possui atributos específicos. 

A classe possui membros como matrícula, salário, departamento, carga horária e data de ingresso. Além disso, herda atributos e métodos da classe \doxylink{class_pessoa}{Pessoa}.
